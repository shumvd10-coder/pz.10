TableNow — умное бронирование столиков 🍽️

Сервис для бронирования столиков в ресторанах с удобным интерфейсом, фильтрами и мгновенным подтверждением.

TableNow — это многостраничный сайт, разработанный в рамках учебной практики. Проект представляет собой лендинг с дополнительными страницами контактов и FAQ, выполненный с использованием CSS-фреймворка Foundation.
